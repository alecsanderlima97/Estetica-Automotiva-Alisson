# Aesthetic Auto System - Sistema de Estética Automotiva

Este é o sistema inovador desenvolvido para a gestão completa de serviços de estética automotiva, utilizando como base sólida a arquitetura do Valen-Studio.

## 🚀 Estrutura Modular
O projeto foi redesenhado em módulos independentes para garantir que modificações em uma área não afetem o funcionamento global:

- **Frontend (`src/components`, `src/pages`)**: Interface rica e interativa para o usuário final e administradores.
- **Workflow de Negócio (`src/services`)**: Fluxo estruturado de agendamento, avaliação técnica e status de serviço.
- **Design System (`src/styles`)**: Identidade visual premium, com foco em estética automotiva de alto padrão.
- **Backend (API/Interface de Dados)**: Camada responsável pela persistência e lógica de processamento.

## ⚙️ Tecnologias
- React + Vite
- Lucide React (Ícones Modernos)
- Vanilla CSS (Aesthetics Premium & Dynamic Animations)
- State Management for Business Workflows

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
